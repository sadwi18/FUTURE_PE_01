# FUTURE_PE_01 – AI Website Copy Generator for Local Businesses

## Business Chosen
Drape & Glow – A traditional yet premium beauty studio located in Turupu Peta, near Ramalayam, Ramavaram (533264).

## Target Audience
Brides and women aged 18–40.

## Objective
To build a structured prompt system that generates high-quality, conversion-focused website copy for local beauty businesses.

## Problem Identified
Many local beauty studios:
- Use unclear website messaging
- Lack strong bridal positioning
- Have weak booking calls-to-action

## Prompt Engineering Approach
The system was designed using:
- Structured business input fields
- Clear output formatting rules
- Tone control mechanisms
- Benefit-driven writing constraints
- Value proposition refinement logic

This ensures:
- Reusable prompt framework
- Business-specific content
- Non-generic AI output

## Deliverables Included
- Homepage copy
- Services page content
- CTA sections
- Tone adaptation system
- Value proposition refinement system

## Tools Used
- ChatGPT

- GitHub
- Lovable

This project was designed as a real client-ready website copy solution.
